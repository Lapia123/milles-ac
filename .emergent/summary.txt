<analysis>**original_problem_statement:**
The user wants to build a back-office accounting software for an FX (foreign exchange) broker.

**PRODUCT REQUIREMENTS:**
- **User Roles:** Admin, Sub-admin, Accountant, and Vendor for the back office.
- **Authentication:** JWT (email/password) and Google Social Login.
- **Client Management:** Create, Read, Update, Delete (CRUD) for clients, including KYC status, MT5 Number, CRM Customer ID, and saved bank account details.
- **Transaction Management:**
    - Track deposits, withdrawals, and transfers linked to a client.
    - Support uploading proof of payment.
    - Specify a destination: Treasury/Bank, PSP, Vendor, Client's own Bank Account, or USDT address.
    - Multi-currency support, with all final balances tracked in USD.
- **Treasury Management:** Manage bank and USDT wallet accounts. Show total treasury balance in USD, transaction history, filtering, statement downloads, and support for inter-treasury transfers.
- **Accountant Dashboard:** A dedicated view for accountants to approve or reject pending client transactions and settlements, protected by a math captcha. For withdrawals, accountants must select a source treasury account and upload proof of payment.
- **PSP (Payment Service Provider) Management:** A manual ledgering system to track PSPs.
- **Vendor Portal:** A separate portal for vendors to manage their assigned transactions.
- **Settlement Approvals:** Admin-initiated settlements for vendors require final approval from an accountant.

**User's preferred language**: English

**what currently exists?**
A full-stack application with a React frontend and a monolithic FastAPI backend using a MongoDB database.
- **Backend ():** Contains all API endpoints for users, clients, transactions, treasury, PSPs, and vendors. It handles auth, complex transaction/settlement workflows, role-based access, client bank account management, and inter-treasury transfers.
- **Frontend:** A React SPA with pages for all major features. The UI uses TailwindCSS and Shadcn/UI.
- **Key Features Implemented:**
    - Authentication (JWT & Google).

    - Treasury management with support for Bank and USDT wallet accounts, including transaction history and filtering.
